## About
This project was largely about understanding the use of bundlers, configuring Webpack and setting up its development server.

## Getting started

1. Clone the repository to your local machine. 
2. Run `npm install` in the project directory to install the required dependencies. 
3. Start the website with `npx webpack serve`.
4. Open `http://localhost:8080` in your browser.
